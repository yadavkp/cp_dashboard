# CP Dashboard

A simple Codeforces dashboard for searching and comparing competitive programming profiles.

## 🚀 What it does

- Search a Codeforces user by handle
- Display profile details: rating, max rating, rank, solved problems
- Show contest history in a table
- Render a rating graph for the selected user
- Compare multiple users in one overlapping rating chart
- Toggle between light and dark theme

## 🧰 Tech Stack

- HTML
- CSS
- JavaScript
- [Chart.js](https://www.chartjs.org/)
- [Codeforces API](https://codeforces.com/api/help)

## 📍 Live file

[CP Dashboard index.html](https://github.com/yadavkp/cp_dashboard/blob/main/cp-dashboard/index.html)

## ✅ How to use

1. Open `cp-dashboard/index.html` in your browser
2. Enter a Codeforces username and click **Analyze**
3. View profile details, solved count, and contest history
4. Click **Compare Users** to add more users
5. See overlapping rating graphs for selected users
6. Click **Toggle Theme** to switch modes

## 💡 Notes

- The `Solved` count is calculated from unique accepted submissions
- The rating graph uses Codeforces contest history
- Multi-user comparison draws each user as a separate colored line
